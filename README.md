## 📖 关于本项目

本项目是 Learn to Read Latin (Andrew Keller & Stephanie Russell, Yale UNIVERSITY PRESS, 2nd Edition) 的中文非官方翻译。

**翻译目的**：
- 辅助个人语言学习，加深对原文的理解
- 为中文读者提供一份参考性中译

**⚠️ 重要声明**：
- 本翻译**仅供学习参考**，不用于任何商业用途
- 版权归原作者及出版社所有
- 若原出版社要求下架，本项目将立即删除
