## 音节

| 概念    |                                        |
| ----- | -------------------------------------- |
| 末音节   | 倒数第一个音节                                |
| 次音节   | 倒数第二个音节                                |
| 前音节   | 倒数第三个音节                                |
| 自然长音  | 含有一个长元音或双元音的音节                         |
| 位置长音  | 音节里一个短元音后跟两个或多个辅音                      |
| 次音节法则 | 如果次音节是长音节，则重读次音节。<br>如果次音节是短音节，则重读前音节。 |
