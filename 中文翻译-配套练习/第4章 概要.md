# 第4章 概要

## 动词词法

|     | 主动态人称词尾    |          | 被动态人称词尾        |           |
| --- | ---------- | -------- | -------------- | --------- |
|     | *单数*       | *复数*     | *单数*           | *复数*      |
| 1   | **-ō, -m** | **-mus** | **-or, -r**    | **-mur**  |
| 2   | **-s**     | **-tis** | **-ris / -re** | **-minī** |
| 3   | **-t**     | **-nt**  | **-tur**       | **-ntur** |

**长元音规则**：长元音在 **-m, -t, -nt, -r, -ntur** 前面会变成短元音。

| 直陈式      | 第三变位法                               | 第三变位法 i-词干                          | 第四变位法                       |
| -------- | ----------------------------------- | ----------------------------------- | --------------------------- |
| **现在时**  |                                     |                                     |                             |
|          | 取现在词干（去掉 **-re**）                   | 取现在词干（去掉 **-re**）                   | 取现在词干（去掉 **-re**）           |
|          | **-e- → -i-** (**rege- → regi-**)   | **-e- → -i-** (**cape- → capi-**)   | NO CHANGE (**audi-**)       |
|          | 加上适当的（主动或被动）人称词尾                    | 加上适当的（主动或被动）人称词尾                    | 加上适当的（主动或被动）人称词尾            |
| **未完成时** |                                     |                                     |                             |
|          | 取现在词干（去掉 **-re**）                   | 取现在词干（去掉 **-re**）                   | 取现在词干（去掉 **-re**）           |
|          | **-e- → -iē-** (**rege- → regiē-**) | **-e- → -iē-** (**cape- → capiē-**) | **-ī- → -iē-** (**audiē-**) |
|          | 加上 **-bā-**                         | 加上 **-bā-**                         | 加上 **-bā-**                 |
|          | 加上适当的（主动或被动）人称词尾                    | 加上适当的（主动或被动）人称词尾                    | 加上适当的（主动或被动）人称词尾            |
| **将来时**  |                                     |                                     |                             |
|          | 取现在词干（去掉 **-re**）                   | 取现在词干（去掉 **-re**）                   | 取现在词干（去掉 **-re**）           |
|          | **-e- → -iē-** (**rege- → regiē-**) | **-e- → -iē-** (**cape- → capiē-**) | **-ī- → -iē-** (**audiē-**) |
|          | 加上适当的（主动或被动）人称词尾                    | 加上适当的（主动或被动）人称词尾                    | 加上适当的（主动或被动）人称词尾            |

**例外**
- 现在时：
	- 一单：**regō, regor** (而非 \*regiō, regior)
	- 二单：**regeris/regere** (而非 \*regiris/\*regire); **caperis/capere** (而非 \*capiris/\*capire)
	- 三复：**regunt, reguntur** (而非 \*regint, \*regintur); **capiunt, capiuntur** (而非 \*capint, \* capintur); **audiunt, audiuntur** (而非 \*audint, \*audintur)
- 将来时：
	- 一单：**regam, regar** (而非 \*regem, \*reger); **capiam, capiar** (而非 \*capiem, \*capier); **audiam, audiar** (而非 \*audiem, audier)

**现在时被动态不定式**: 把第二主要部分末尾的 **-e** 换成 **-ī-**。在第三变位法中把末尾的 **-ere** 换成 **-ī-**。

| 命令式     | 现在时主动态                 |                           | 现在时被动态         |                        |
| ------- | ---------------------- | ------------------------- | -------------- | ---------------------- |
|         | *单数*                   | *复数*                      | *单数*           | *复数*                   |
|         | 现在词干                   |                           | 现在词干 + **-re** | 现在词干 + **-minī**       |
| **例外**: | **dīc, dūc, fac, fer** | **regite, capite, ferte** |                | **regiminī, capiminī** |

[^2]: **meus** 的阳性单数呼格是 **mī**.

## 名词句法

| 概念                                                             | 示例                                                              |
| -------------------------------------------------------------- | --------------------------------------------------------------- |
| **部分属格**<br>表示一个 *整体*，另一名词是这个整体的一部分，从属于这个整体                    | **Multī *agricolārum* nōn labōrābant.**<br>*农民* 中的许多人当时不在工作。    |
| **主语属格**<br>表示 *执行另一名词所隐含的动词动作* 的人或事物                          | **Magnum erat odium *Sullae* in Rōmānōs.**<br>*苏拉* 对罗马人的仇恨是巨大的。 |
| **宾语属格**<br>1. 表示 *接受另一名词所隐含的动词动作* 的人或事物<br>2. 可用英文"for"翻译     | **Rēgīna odium *populī* habet.**<br>女王怀有对 *人民* 的仇恨。             |
| **方面夺格**<br>1. *限定* 或 *具体说明* 形容词、动词或句子的含义  <br>2. *不使用* 介词<br> | **Magnus *cōnsiliō* erat poēta.**<br>那位诗人 *在判断力方面* 很出色。         |
