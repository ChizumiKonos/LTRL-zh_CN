# 第 3 章 概要

## 第一和第二变格法形容词：词尾

|     | *单数*          |         |         | *复数*      |           |           |
| --- | ------------- | ------- | ------- | --------- | --------- | --------- |
|     | M.            | F.      | N.      | M.        | F.        | N.        |
| 主   | **-us/—**[^1] | **-a**  | **-um** | **-ī**    | **-ae**   | **-a**    |
| 属   | **-ī**        | **-ae** | **-ī**  | **-ōrum** | **-ārum** | **-ōrum** |
| 与   | **-ō**        | **-ae** | **-ō**  | **-īs**   | **-īs**   | **-īs**   |
| 宾   | **-um**       | **-am** | **-um** | **-ōs**   | **-ās**   | **-a**    |
| 夺   | **-ō**        | **-ā**  | **-ō**  | **-īs**   | **-īs**   | **-īs**   |
| 呼   | **-e**        |         |         |           |           |           |
从阴性单数主格形式中去掉 **-a** 获取词干。

| 语法概念                                                                                                     | 示例                                                                                                                |
| -------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| **名词-形容词一致**<br>形容词必须要与它修饰的名词在 *性、数、格* 上一致。                                                              | **nauta malus** (名词和形容词均为阳性单数主格)                                                                                  |
| **形容词的名词化用法**<br>1. 用作名词的形容词叫作 *名词化形容词*。<br>2. 名词化形容词的翻译必须体现出性和数。                                        | ***Bonus multa* videt.**<br>*那个好人* 看到 *很多东西*。                                                                     |
| **表语形容词**<br>1. 与系动词一同出现<br>2. 总是与它修饰的名词在 *性、数、格* 上一致。                                                   | ***Clārus* est poēta.**<br>这位诗人很*有名*。                                                                             |
| **主语不定式**<br>不定式（一种中性单数的动词性名词）可以作为其他动词的 *主语* 。                                                           | **Bonum est *labōrāre*.**<br>*去工作* 是好的。                                                                           |
| **人物施事夺格**<br>1. 表示实施该动词动作的人物或施事者。<br>2. 常与 *被动* 态动词连用<br>3. 需要 **Ā/AB**                                 | **Fīlius *ā regīnā* vocābātur.**<br>儿子 *被女王* 召唤。                                                                  |
| **方式夺格**<br>1. 表示动作实施的方式或方法<br>2. 没有被形容词修饰时需要 **cum**<br>3. 被形容词修饰时可以使用（但不强制需要）**cum**<br>4. 可翻译成副词“……地” | ***Cum dīligentiā* labōrat.**<br>她 *带着勤奋* 工作。<br>***Magnā (cum) dīligentiā* labōrat.**<br>她 *带着很大的勤奋 / 非常勤奋地* 工作。 |
| **同位语**<br>1. 定义或限定与其相邻的名词<br>2. 总是与其定义的名词在 *格* 上一致                                                      | **Horātius, *poēta* magnus, multōs librōs habet.**<br>贺拉斯，一位大*诗人*，有很多书。                                           |

[^1]: 有些形容词 *没有* 阳性单数主格词尾（例如 **miser**, **pulcher**）。阳性单数主格必须要当作词汇条目的一部分去记忆。

[^2]: 例外：**vocor** (而非 \*vocāor); **moveor** (而非 \*movēor).

[^4]: 例外：**vocāberis/vocābere** (而非 \*vocābiris/\*vocābire); **movēberis/movēbere** (而非 \*movēbiris/\*movēbire).
[^5]: 例外：**vocābuntur** (而非 \*vocābintur); **movēbuntur** (而非 \*movēbintur).
