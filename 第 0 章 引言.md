# 引言 
## 拉丁语 

拉丁语属于印欧语系。"印欧"这一名称表明了这些语言最初使用的地理区域。该语系包括欧洲使用的大多数语言，以及远至古代波斯、阿富汗和印度等东方地区所使用的语言。通过对词汇、形态和句法的细致比较，学者们已经证明，所有这些语言都源自一个共同的祖先，这个祖先被称为**印欧语**（Indo-European, **IE**）或**原始印欧语**（Proto-Indo-European, **PIE**），其使用时间大约在公元前五千年（见图1）。据推测，随着时间的推移，使用这种原始语言的人群逐渐分散到欧洲、亚洲和印度各地，语言在不同地区发生了不同的变化，最终形成了属于这一语系的多种语言。

关于PIE或使用该语言的人群，没有直接的证据——即文字或考古证据——留存下来。对于这种语言的了解，都来源于对现存语言的比较研究。对这些语言的研究始于18世纪末，当时威廉·琼斯（William Jones）爵士，一位律师兼东方语言研究者，首次公开断言希腊语、拉丁语和古印度梵语源自一个共同的源头。对印欧语言的科学研究则始于19世纪早期弗朗茨·博普（Franz Bopp）的著作。他比较了拉丁语、希腊语、梵语、古波斯语以及日耳曼语族（英语即属于该语族）中的动词形式。

印欧语言经过分析被划分为不同的子群或方言，而拉丁语属于其中一个被称为**意大利语族**（Italic）的子群，因为该子群的语言都曾在意大利半岛使用。与PIE一样，意大利语族也没有留下直接的证据，但属于这一语群的四种语言——**拉丁语**（Latin）、**法利斯坎语**（Faliscan）、**奥斯坎语**（Oscan）和**翁布里亚语**（Umbrian）[^1]——彼此之间的关系，比它们与其他印欧语言的关系更为密切，因此被认为源自一个单一的源头。

[^1]: 法利斯坎语在意大利的伊特鲁里亚（Etruria）地区（意大利中部）使用，奥斯坎语在坎帕尼亚（Campania）地区（意大利南部）使用，翁布里亚语在翁布里亚（Umbria）地区（意大利中北部）使用。早期与罗马发生冲突的古意大利民族萨姆尼特人（Samnites）讲的就是奥斯坎语。

拉丁语的早期使用者被称为*拉丁人*（Latīnī）。在公元前一千年早期，他们就已生活在意大利中部的许多定居点中。罗马只是这些定居点之一。虽然罗马的早期历史鲜有定论，但传统上认为其建国于公元前753年，而近期的考古发掘为这一早期年代提供了些许佐证。传统还记载，罗马在其存在的最初250年里由伊特鲁里亚血统的国王统治。在公元前3世纪罗马崛起之前，伊特鲁里亚文明一直主导着意大利中部半岛。拉丁语从伊特鲁里亚语中借用了许多词汇，而伊特鲁里亚语与原始印欧语*并无*关联。

流传至今最早的拉丁语书面证据以铭文形式出现，这些铭文的年代可追溯至公元前七世纪末和六世纪初，其上的语言可辨认出是拉丁语，但与数百年后罗马鼎盛时期所书写和使用的拉丁语大相径庭。这些铭文来自意大利中部的许多城镇，而不仅仅是罗马。关于该时期拉丁语的其他证据，则来自后世拉丁语作家对更早时期语言的引述。

根据罗马传统，伊特鲁里亚人以君主制统治罗马，直至公元前509年，罗马人推翻了他们的外族统治者并建立了共和国。权力转移到每年选举产生的两名被称为执政官（consuls）的官员手中，他们与元老院共同掌握着罗马的政治权力。在接下来的几个世纪里，罗马卷入了一外一内两场冲突。对外，罗马人几乎不断地与意大利半岛的其他民族以及地中海的其他强国交战，最著名的是以北部非洲为基地的迦太基人。对内，罗马人则挣扎于下层平民（plebs）与统治他们的贵族家族之间的内部冲突。尽管内外受挫，然而罗马还是通过条约与征服逐步扩大了其影响力，到公元前3世纪末，它已控制了现今意大利的大部分地区，并同时向地中海东部挺进。

**拉丁文学**首次出现在这一政治扩张时期的后期。这一时期作品得以幸存的最著名作家有喜剧诗人**普劳图斯**（Plautus，公元前254–184年）和**泰伦提乌斯**（Terence，公元前185？–159年）、史诗诗人**恩尼乌斯**（Ennius，公元前239–169年），以及演说家、历史学家和散文家**加图**（Cato，公元前234–149年）。喜剧作家的语言常被认为包含当时拉丁语实际口语方式的证据，因为喜剧中的人物来自各行各业。恩尼乌斯除了拉丁语外，还会说希腊语和奥斯坎语，他借鉴并改编了希腊语的许多特征，是最早用拉丁语创作史诗的作家之一。他的风格极大地影响了后来的诗人，可以说，他是第一个确立了*文学语言*——一种词汇和句法规则日益严格的书面语言——的人。这些作家共同代表了所谓的**早期拉丁语**（early Latin）。

在接下来的两个世纪里，直到公元前一世纪末，在罗马征服希腊后，罗马作家开始接触到希腊文学，他们深受希腊文学的影响，继续发展着拉丁语的文学语言。他们制定了拼写、形态、句法和词汇的规则，并区分了诗歌和散文的正确用法。其结果通常被称为**古典拉丁语**（classical Latin），即在共和国晚期和帝国早期（大约公元前100年至公元100年）[^3]期间，受过教育的罗马民众所书写和口语使用的[^2]语言。创作古典拉丁语的作家中，散文方面有：**凯撒**（Caesar，公元前100–44年）、**西塞罗**（Cicero，公元前106–43年）和**李维**（Livy，公元前59年–公元17年）；诗歌方面有：**卡图卢斯**（Catullus，公元前84？–54？年）、**维吉尔**（Vergil，公元前70–19年）和**贺拉斯**（Horace， 公元前65–8年）。

[^2]: 一些现存文本，例如西塞罗的演说词，表明古典拉丁语既是书面语*也是*口语。 

[^3]: 当然，不同作家之间以及这一时期初期到末期的用法和风格会有所差异。人们仍然习惯提及黄金时代拉丁语（公元前一世纪的作家如凯撒、西塞罗、维吉尔和贺拉斯所使用的拉丁语）和白银时代拉丁语（公元一世纪的作家如塞内加（Seneca）和昆体良（Quintilian）所使用的拉丁语）。 

本书向学生介绍这些作家及同期其他作家所写的语言。书中会指出散文与诗歌之间的用法区别，但在大多数情况下，这里所呈现的古典拉丁语规则可假定适用于整个这一时期的文学。

## 古典拉丁语的发音 

尽管古典拉丁语的发音必然存在如同任何现代语言中明显可见的变异，但有大量古代证据表明其存在一种标准发音，此处呈现的规则即近似于该标准。[^4] 

[^4]: 对于这一发音的呈现，作者们要感谢 W. S. Allen 的 *Vox Latina*（剑桥，1965年）。

古典拉丁语的发音规则是从以下几种证据中重建的： 

1. 拉丁语语法学家和拉丁语作者关于拉丁语发音具体要点的陈述。拉丁语语法学家的活跃时期远晚于古典拉丁语所处的时代，但他们引用了更早的拉丁语作者的作品，并证实了从其他来源得知的发音特征。 
2. 铭文中拉丁语单词的正字法或拼写。古代拼写，无论是拉丁语还是其他古代语言，其标准化程度都远低于现代拼写。拼写的变化通常表明发音的差异。 
3. 拉丁语单词在其他语言中的表示或音译，以及外来词在拉丁语中的表示或音译。特别是人名的音译，提供了关于人名在罗马历史不同时期如何发音的宝贵信息。 
4. 拉丁语的内部语法和诗歌结构。这包括从拉丁诗歌的格律结构中获得的证据。 
5. 拉丁语单词的词源，以及从拉丁语演变而来的罗曼语族各语言中拉丁语单词的发展。

## 字母表

拉丁字母表与英语字母表几乎相同，但没有字母 j 和 w。

**注意事项**

1. **k** 仅用于少数单词，大多源自外来语。**y** 和 **z** 出现在从希腊语借用的姓名及其他词汇的音译中。
2. **i** 既可以是辅音，也可以是元音。在一些拉丁文本中，**j** 被用作辅音 **i** 的符号，但在本书中，**i** 既用于辅音也用于元音。
3. **u** 同样既可以是辅音，也可以是元音。在古典时期（以及之后的几个世纪），罗马人使用字母 **v** 来表示辅音和元音，而 u 只是这个字母的圆体写法。一些现代编辑者用 **u** 来表示辅音和元音，而另一些则保留用 **v** 表示辅音、用 **u** 表示元音的习惯。本书遵循后一种习惯。
4. 大写字母 **C**，最初是希腊字母 *gamma* (**Γ**) 的一种形式，最早被伊特鲁里亚人使用，他们不区分英语中硬音 **g** 和硬音 **c** 的发音。即使在拉丁语发展出一个单独的新字母来表示硬音 **g** 之后，大写字母 **C** 在某些名字的缩写中仍然继续表示硬音 **g**（例如，**C.** = Gaius）。

| 字母  | 发音              | 发音参考          |
| --- | --------------- | ------------- |

**注意**

## 元音

拉丁语有两组五个元音：

| 短元音 | 长元音 |
| --- | --- |
| a   | ā   |
| e   | ē   |
| i   | ī   |
| o   | ō   |
| u   | ū   |

**注意**
1. 元音既有*音长*也有*音质*。*音长*指的是元音的*长度*。拉丁语中带有**长音符号**或**长音符**（ˉ）的元音是*长元音*，发音时长大约是短元音的两倍。没有此标记的元音是*短元音*。*音质*指的是元音的发音方式。例如，**ā** 和 **ō** *音长*相同，但*音质*不同。 
2. 拉丁语单词中的元音*本质*上要么是长元音，要么是短元音。重要的是要认识到，例如，**a** 和长元音 **ā** 是两个不同的元音，即使它们密切相关。 
3. 元音长度的差异常常传达出重要的意义区别。例如，单词 **alium**（带有短音 **a-**）意思是"另一个人"；单词 **ālium**（带有长音 **ā-**）意思是"大蒜"。

**长元音上的长音符号应被视为拉丁单词拼写的一部分。学习新词汇时，请记住所有长音符号，并始终在长元音上标注它们。**[^5]

[^5]: 本书中的所有长元音都用长音符号标出。未标记此符号的元音都是短元音。

元音发音如下：

| 字母  | 发音   | 发音参考                    |
| --- | ---- | ----------------------- |
| a   | /a/  | **a**lert, c**u**p      |
| ā   | /aː/ | f**a**ther              |
| e   | /e/  | p**e**t                 |
| ē   | /eː/ | (?)f**a**te             |
| i   | /i/  | f**i**t                 |
| ī   | /iː/ | f**ee**t                |
| o   | /o/  | (?)s**o**ft, c**au**ght |
| ō   | /oː/ | (?)h**o**pe             |
| u   | /u/  | p**u**t                 |
| ū   | /uː/ | f**oo**l                |
译注：(?)表示存疑

**注意**
1. **a** 和 **ā** 在音质上非常相似，仅在音长上不同。必须注意这两个元音的发音。所有其他元音对（e 和 **ē** 等）在音质*和*音长上*都*不同，说英语的人更容易区分。（译注：存疑） 
2. **y** 是一个元音，代表希腊语的 *upsilon* (**Y**)。这个元音发音类似法语的 **u**（介于英语 **i** 和 **u** 之间的一个音）（译注：就是汉语的 ü）。它可以是长音也可以是短音；长元音的发音时长是短元音的两倍。

## 双元音

双元音是由*两个元音连读*而产生的一个*单一的*元音音素。（单词 *diphthong* 在希腊语中意为"双重声音"。）拉丁语中有六个双元音，其发音如下：

| 字母  | 发音   | 发音参考     |
| --- | ---- | -------- |
| ae  | /ai/ | h**i**gh |
| oe  | /oi/ | b**oy**  |
| ei  | /ei/ | d**ay**  |
| ui  | /ui/ | t**wi**n |
| au  | /au/ | h**ow**  |
| eu  | /eu/ |          |

**注意**
- 没有与拉丁语双元音 **eu** 对应的英语发音。正如其拼写所示，它是短元音 **e** 和 **u** 的组合，并作为一个音发出。如果 Elmer Fudd 发 "very"（v**ew**y），其中的 **ew** 会非常接近于拉丁语 **eu** 的音。

## 重音

拉丁语有重音。这意味着单词发音时，其中一个音节会稍微*重读*或强调。要确定哪个音节需要重读，必须首先将拉丁单词划分成音节。

### 音节划分

音节是在一次呼气冲动中发出的一个或一连串声音。**一个拉丁单词有多少个元音或双元音，就有多少个音节。拉丁语中没有不发音的元音或辅音。**

划分单词音节时，

1.  单个辅音与其后的元音划归为一组：
	- anima → a/ni/ma
	- aurum → au/rum
	- gladius → gla/di/us
	- vitaque → vi/ta/que

	**注意** 
	- **qu** 总被视为一个*单*辅音。**su** 和 **gu** 有时被视为单辅音。

2.  如果有两个或两个以上的辅音连续出现，则最后一个辅音与其后的音节归为一组：

    - imperium → im/pe/ri/um
    - sānctus → sānc/tus
    - virumque → vi/rum/que
    - puella → pu/el/la

3.  然而，如果两个辅音的组合是一个塞音（c, k, t, p, ch, th, ph, g, d, b）或摩擦音 f 后接一个流音（l, r），则这两个辅音保持在一起：[^6]

    - patria → pa/tri/a
    - agricola → a/gri/co/la
    - ablātus → a/blā/tus

[^6]: 发音时需要停顿气流的辅音称为**塞音**。例如，发 **g** 音时，喉咙收缩，气流停止。同样，发 **p** 音时，双唇闭合，气流也停止。**摩擦音**是通过迫使气流通过狭窄通道而产生的辅音。**l** 和 **r** 被称为**流音**，因为它们的声音可以像元音一样被延长。

### 重音规则

拉丁语单词中的*最后一个*音节被称为**尾音节**（ultima，< **syllaba ultima**，即“最后一个音节”）。从末尾数*第二个*音节被称为**次音节**（penult，< **syllaba paenultima**，即“几乎最后一个音节”）。从末尾数*第三个*音节被称为**前音节**（antepenult，< **syllaba antepaenultima**，即“倒数第二音节之前的音节”）。 

**拉丁语单词只有*次音节*或*前音节*可以被重读**。*如果一个单词只有两个音节，则重读次音节。* 以下单词因为*只有*两个音节，所以重读在次音节上。请大声练习发音，重读次音节。

-   **ta**men
-   **cōn**sul
-   **mū**tat
-   **o**pus

如果一个单词有两个以上的音节，其重音根据称为**次音节法则**的规则确定：

> 如果次音节是长音节，则重读次音节。如果次音节是短音节，则重读前音节。

一个音节在包含以下任意一项内容时是长音节：

1.  一个长元音或双元音（**自然长音**）
2.  一个短元音*后跟*两个或两个以上的辅音（*不一定在同一个音节中*）（**位置长音**）

以下单词因为其次音节是自然长音，所以重音在次音节。请大声练习发音，重读次音节。

## 语言学习：词汇、词法与句法

基础语言学习可分为三个部分：词汇、词法和句法。

### 词汇

词汇首先指习得另一种语言中词语的含义。例如，拉丁词 **puella** 意为"女孩"；**superō** 意为"克服、征服"。除含义外，词汇还包括其他重要元素，提供关于（名词的）性、词干和词类的关键信息。*完整的词汇条目*（即词语在词典中的呈现方式）包含所有这些信息。请务必熟记每个新学词汇的完整词汇条目；仅记住含义是远远不够的。
### 词法

词法指研究词语在特定语言中所呈现的不同*形式*。例如，在英语中，必须学习动词的各种形式：

1.  大多数英语动词的第三人称单数现在时形式是通过添加 **-s** 或 **-es** 构成的：

    - walk > walk**s**
    - go > go**es**

2.  许多英语动词的过去式是通过添加 **-ed** 构成的：

    - walk > walk**ed**

词法也延伸到名词、形容词以及其他某些具有多种形式的词类。*通过改变形式来传达变化的意义*被称为**屈折变化**。

有些语言比其他语言具有更丰富的屈折变化。因此，英语名词通常只以四种形式出现：

- boy（单数） 
- boys（复数） 
- boy's（单数所有格）
- boys'（复数所有格）

相比之下，拉丁语名词通常以十种或更多形式出现。由于拉丁语是一种屈折变化比英语更丰富的语言，所以拉丁语的词法比英语词法更广泛，需要完全掌握。

### 句法

*词语在句子中的语法功能*就是该词的**句法**。在英语句子“The woman gives the cat a toy”中，“woman”、“cat”和“toy”都具有不同的语法功能，这些功能共同传达句子的完整含义：

- woman: 主语
- cat: 间接宾语
- toy: 直接宾语

**主语**、**间接宾语**和**直接宾语**这些术语是英语句法的例子，这些术语以及许多其他英语句法术语也用于拉丁语句法中。准确翻译拉丁语句子需要了解该句中各词语的句法。在本书中，新的拉丁语句法术语将会被陆续引入和定义。这些术语在首次出现时总是以粗体显示。请学会所有粗体显示的句法术语。
