# 前言

*Learn to Read Latin*（以下简称 LTRL）是一本集入门语法与读本于一体的教材。本书旨在帮助学生尽快获得阅读和欣赏拉丁文学伟大作品的能力。本书尊重教师和学生，假定学习者对学好、学透拉丁语怀有浓厚的兴趣。

LTRL 源于一个诚挚的愿望，即让其他人也能分享我们两人在阅读拉丁语原版文学时所体验到的那种巨大乐趣。多年来，在我们看来，人们过于强调教学生翻译拉丁语，而即使是最勤奋的拉丁语学生，对拉丁语言及伟大拉丁作家的风格也知之甚少。许多拉丁语入门教材要求学生翻译几卷用拉丁语写成的故事，而这些故事并非出自古代作家之手，而是教材作者自己编写的。在我们看来，如果目标是培养学生阅读和欣赏凯撒、西塞罗、李维、维吉尔、贺拉斯以及其他众多作家的作品，那么这种做法是对学生精力的低效使用，其教学方法的价值也十分有限。

此外，我们注意到，入门教材常常提供关于罗马家庭生活和文化的一些零碎信息，却未能向学生介绍那些随着他们语言能力提高而能够接触到的众多作家和文学作品。根据我们的经验，这意味着许多学生在入门阶段后选择不再继续学习，部分原因是他们不知道在中级和高级阶段可以阅读什么。我们试图编写一本主要依靠古代作家本身的拉丁语入门教材，以此作为学生学习拉丁语句法和风格的途径。我们力求向学生介绍拉丁文学，这对许多人来说将成为他们日后学习的重点，并且我们坚信，了解罗马文化的最佳途径，就是尽可能广泛地阅读来自各种不同作家的罗马文学作品。

LTRL 深受两本教材的影响，它们是为纽约市立大学布鲁克林学院拉丁语/希腊语研究所编写的：《拉丁语：强化教程》（*Latin: An Intensive Course*），作者是弗洛伊德·L·莫兰（Floyd L. Moreland）和丽塔·M·弗莱舍（Rita M. Fleischer）；以及《希腊语：强化教程》（*Greek: An Intensive Course*），作者哈迪·汉森（Hardy Hansen）和杰拉尔德·M·奎因（Gerald M. Quinn）。这两本书都是为快节奏的暑期课程设计的，每天覆盖一个新的单元。它们都是清晰、高效和完整的典范。弗洛伊德·莫兰是我们的老师和同事，我们从他那学到了语法和词形变化知识点呈现顺序的重要性，以及简洁清晰解释的价值。他将他个人对拉丁语句法和风格的浓厚兴趣传递给了我们两人。他教导我们在阅读和翻译时要循序渐进，要关注最微小的细节，并热爱拉丁文学。他强大的影响力体现在我们书中许多优秀之处。我们从哈迪·汉森和格里·奎因那里借鉴了在词形和句法讲解之后放置编号注释的做法，并仿照他们的词汇注释编写了我们自己的词汇注释，像他们在希腊语教材中所做的那样，包含了相当数量的语言学信息。

由于 LTRL 并非为满足强化暑期课程的特殊要求而编写，我们的书在几个重要方面与其前身有所不同。各章节篇幅更长，并呈现完整的系统或密切相关的词汇、词形和句法元素。练习册中提供了大量练习和练习句，供课堂使用、家庭作业以及针对个别学生的额外补习和辅导。书中对英语语法和用法的某些要点进行了较详细的解释，并为此提供了相应的练习。有两个目标决定了LTRL 中语法知识点的呈现顺序。我们希望让学生感受到拉丁语句法发展的逻辑脉络，并获得逐步深化的理解。我们也希望提供尽可能早的机会，让学生接触到未经删节的拉丁语原文段落。

LTRL 可能是同类教材中第一本广泛使用帕卡德人文学院（Packard Humanities Institute）CD-ROM的书籍，该光盘包含了直至查士丁尼时期的大部分拉丁文学语料库。借助这一强大工具，我们常能为练习和练习句确立正确的拉丁语用法，并尽力排除任何在现存古典拉丁语中未出现的短语或句法结构。虽然可能仍存在一些用法错误，但我们相信，我们书中自行撰写的拉丁语部分与学生在古代作家那里将会读到的内容相似。我们还利用该CD-ROM找到了许多能很好地说明书中每章所呈现的词汇、词形和句法的段落。LTRL 中广泛的作家和文本范围——包括一些不常被阅读的作家的精选段落——部分地得益于这张CD-ROM。在本书的编写过程中，我们定期参考了几本拉丁语法书和词典，特别倚重乔治·M·莱恩（George M. Lane）的《拉丁语法》（*A Latin Grammar*）和E.C.伍德科克（E. C. Woodcock）的《新拉丁语句法》（*A New Latin Syntax*）。关于拉丁词中的元音音量，我们参考了瓦尔德（Walde）和霍夫曼（Hofmann）的《拉丁语词源词典》（*Lateinisches Etymologisches Wörterbuch*）。

## 教科书第二版前言

LTRL 于2003年首次出版。现在我们很高兴有机会根据第一版使用者提出的意见和建议，出版修订版。我们感谢他们的帮助。全书进行了多处修正。对词汇注释以及句法和词法部分进行了编辑和重写，旨在简化和明晰讲解内容。第十四章新增了一节关于不定代词和不定形容词的内容（此前仅在词汇注释中呈现）。有三章（第八章、第九章和第十章）进行了实质性修改，其内容也重新编排，以使讲解更加均衡。一些词汇被移到了不同的章节，并增加了一些新词。每章的短篇和长篇阅读材料都有增补、删节和修正。特别是第二章至第六章，现在提供了更多实际拉丁语的样本，以挑战和激励初学者。所有阅读材料提供的词汇现在均按单词在阅读材料中出现的顺序排列，而非按字母顺序。
