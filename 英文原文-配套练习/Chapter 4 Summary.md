# Chapter 4 Summary

## Verb Morphology

|     | Active Personal Endings |          | Passive Personal Endings |           |
| --- | ----------------------- | -------- | ------------------------ | --------- |
|     | *Sing.*                 | *Plural* | *Sing.*                  | *Plural*  |
| 1   | **-Ňć, -m**              | **-mus** | **-or, -r**              | **-mur**  |
| 2   | **-s**                  | **-tis** | **-ris / -re**           | **-minńę** |
| 3   | **-t**                  | **-nt**  | **-tur**                 | **-ntur** |

**Long Vowel Rule**: LONG VOWELS SHORTEN BEFORE **-m, -t, -nt, -r, -ntur**.

|            | Present Indicative                      | Stem change                       |                                                      |
| ---------- | --------------------------------------- | --------------------------------- | ---------------------------------------------------- |
| 3rd        | Take present stem (by dropping **-re**) | **-e- ‚Üí -i-** (**rege- ‚Üí regi-**) | Add appropriate (active or passive) personal endings |
| 3rd i-stem | Take present stem (by dropping **-re**) | **-e- ‚Üí -i-** (**cape- ‚Üí capi-**) | Add appropriate (active or passive) personal endings |
| 4th        | Take present stem (by dropping **-re**) | NO CHANGE (**audi-**)             | Add appropriate (active or passive) personal endings |

**Exceptions**
- 1st sing.: **regŇć, regor** (NOT \*regiŇć, regior)
- 2nd sing.: **regeris/regere** (NOT \*regiris/\*regire); **caperis/capere** (NOT \*capiris/\*capire)
- 3rd pl.: **regunt, reguntur** (NOT \*regint, \*regintur); **capiunt, capiuntur** (NOT \*capint, \* capintur); **audiunt, audiuntur** (NOT \*audint, \*audintur)

|            | Imperfect Indicative                    | Stem change                         |              |                                                      |
| ---------- | --------------------------------------- | ----------------------------------- | ------------ | ---------------------------------------------------- |
| 3rd        | Take present stem (by dropping **-re**) | **-e- ‚Üí -ińď-** (**rege- ‚Üí regińď-**) | Add **-bńĀ-** | Add appropriate (active or passive) personal endings |
| 3rd i-stem | Take present stem (by dropping **-re**) | **-e- ‚Üí -ińď-** (**cape- ‚Üí capińď-**) | Add **-bńĀ-** | Add appropriate (active or passive) personal endings |
| 4th        | Take present stem (by dropping **-re**) | **-ńę- ‚Üí -ińď-** (**audińď-**)         | Add **-bńĀ-** | Add appropriate (active or passive) personal endings |

|            | Future Indicative                       | Stem change                         |                                                      |
| ---------- | --------------------------------------- | ----------------------------------- | ---------------------------------------------------- |
| 3rd        | Take present stem (by dropping **-re**) | **-e- ‚Üí -ińď-** (**rege- ‚Üí regińď-**) | Add appropriate (active or passive) personal endings |
| 3rd i-stem | Take present stem (by dropping **-re**) | **-e- ‚Üí -ińď-** (**cape- ‚Üí capińď-**) | Add appropriate (active or passive) personal endings |
| 4th        | Take present stem (by dropping **-re**) | **-ńę- ‚Üí -ińď-** (**audińď-**)         | Add appropriate (active or passive) personal endings |
**Exceptions**
- 1st sing.: **regam, regar** (NOT \*regem, \*reger); **capiam, capiar** (NOT \*capiem, \*capier); **audiam, audiar** (NOT \*audiem, audier)

**Present Passive Infinitive**: Change final **-e** of 2nd prin. part to **-ńę-**. In 3rd conjugation change final **-ere** to **-ńę-**.

| Imperative      | Present Active Sing.   | Present Active Pl.        | Present Passive sing.  | Present Passive pl.      |
| --------------- | ---------------------- | ------------------------- | ---------------------- | ------------------------ |
|                 | Present Stem           |                           | Present Stem + **-re** | Present Stem + **-minńę** |
| **Exceptions**: | **dńęc, dŇęc, fac, fer** | **regite, capite, ferte** |                        | **regiminńę, capiminńę**   |

## Noun Syntax

| Concept                                                                                                                                                   | Example                                                                                           |
| --------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------- |
| **Partitive Genitive**<br>REPRESENTS A *WHOLE* OUT OF WHICH ANOTHER NOUN IS A PART                                                                        | **Multńę *agricolńĀrum* nŇćn labŇćrńĀbant.**<br>Many *of the farmers* were not working.                |
| **Subjective Genitive**<br>EXPRESS THE PERSON OR THING *PERFORMING THE VERBAL ACTION IMPLIED IN ANOTHER NOUN*                                             | **Magnum erat odium *Sullae* in RŇćmńĀnŇćs.**<br>Great was the hatred *of Sulla* against the Romans. |
| **Objective Genitive**<br>1. EXPRESS THE PERSON OR THING *RECEIVING THE VERBAL ACTION IMPLIED IN ANOTHER NOUN*<br>2. MAY BE TRANSLATED WITH ENGLISH "FOR" | **Rńďgńęna odium *populńę* habet.**<br>The queen has hatred *of (for) the people*.                   |
| **Ablative of Respect**<br>1. *LIMITS* OR *SPECIFIES* THE MEANING OF AN ADJECTIVE OR VERB OR SENTENCE<br>2. *NO* PREPOSITION USED                         | **Magnus *cŇćnsiliŇć* erat pońďta.**<br>The poet was great *in (respect to) judgement*.              |
